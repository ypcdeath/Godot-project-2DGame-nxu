# Godot-2D-demo-nxu
- 暑假时使用Godot跟着timothyqiu大佬学习制作的2D游戏项目demo。
- (During the summer vacation, I used Godot to learn game development from the master Timothyqiu and completed a 2D game project.)

## 操作指南
- j     - 攻击
- k     - 滑铲
- wasd  - 移动
- space - 跳跃
- e     - 互动
- esc   - 菜单
